# LLM-memory-extractor


A conversational AI system that extracts user preferences, emotional patterns, and facts from chat conversations, with customizable personality modes.

## Description

This project demonstrates memory extraction from chat conversations and personality-adaptive responses. After 10 messages, the system analyzes the conversation and extracts structured data including user preferences, emotional patterns, and key facts. The AI can respond in different personalities like Calm Mentor, Witty Friend, or Therapist.

## Features

- Memory extraction after 10 chat messages
- 7 distinct personality modes
- Structured JSON output for extracted memories
- Personality comparison demo
- Multiple Groq LLM model support

## Requirements

- Python 3.8 or higher
- Groq API key
- Dependencies listed in requirements.txt
